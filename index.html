Office Hours

Office hours are conducted on Mondays, 7pm - 8pm, using Google Hangouts. Join the hangout here. If you experience difficulty joining the hangout, please send an email to buckingham-apcs2014@googlegroups.com, and we will send you a direct invite.

Homework

Assignments can be submitted here. To submit an assignment, please name a file in the submission as follows: Lastname-Week#. The week number should be the week that the assingment was assigned, not the week that it is due. Refer to the week numbers below. If there are multiple files in a submission, only one of them needs to be named this way.
